# SpringModule

## 📖 Overview

SpringModule is a collection of Spring Framework practice modules and examples developed using Java and Maven. This repository contains multiple Spring-based projects demonstrating concepts such as:

* Spring Core
* Dependency Injection (DI)
* Bean Configuration
* Annotation-Based Configuration
* Autowiring
* Bean Scope
* Primary Beans
* Java Configuration
* XML Configuration

The project is mainly designed for learning and practicing Spring Framework concepts from beginner to intermediate level.
